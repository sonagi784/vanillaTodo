# vanilla javascript Todolist

### npm, webpack으로 구조 하나하나 짜기

- webpack 기본구조
- babel, polyfill, eslint, prettier
- jest

### vanilla javascript로 todo-list 만들기

- todolist 진행중/완료 필터링
- todo 추가하기
- todo 추가하기 완료 시 입력창 비우기
- 진행중 리스트 -> 완료된 리스트 이동기능
- 완료된 리스트 -> 복구 기능
- 전체 체크 기능
- 하위 체크박스 해제시 전체 체크박스 자동 해제
- 제목순/최신순 정렬
- 30자 이상 입력 시 alert
- 리스트 저장 로컬스토리지
- 스타일 추가
